# Anjinhos Peludos 🐾

Projeto de extensão universitária desenvolvido para a ONG **Anjinhos Peludos**, com o objetivo de criar uma presença digital que apoie as atividades da organização voltadas ao bem-estar e adoção de animais.

🔗 **Acesse o projeto:** [https://prototipov2-anijinhospeludos.lovable.app/](https://prototipov2-anijinhospeludos.lovable.app/)

---

## Sobre o Projeto

Este sistema é um protótipo desenvolvido como parte de um projeto de extensão de faculdade, em parceria com a ONG **Anjinhos Peludos**. A plataforma tem como objetivo facilitar a divulgação dos animais disponíveis para adoção, bem como apresentar a missão e os valores da organização.

O projeto foi criado utilizando o [Lovable](https://lovable.dev), uma plataforma de desenvolvimento assistido por inteligência artificial.

---

## Tecnologias Utilizadas

- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [React](https://react.dev/)
- [shadcn/ui](https://ui.shadcn.com/)

## Como Executar Localmente

Requisito: ter o [Node.js](https://nodejs.org/) instalado (recomendado via [nvm](https://github.com/nvm-sh/nvm#installing-and-updating)).

```sh
# 1. Clone o repositório
git clone <URL_DO_REPOSITÓRIO>

# 2. Acesse a pasta do projeto
cd prototipov2-anijinhospeludos

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
```

## Como Editar o Projeto

**Via Lovable**

Acesse o projeto no [Lovable](https://lovable.dev) e utilize prompts para realizar alterações. As mudanças feitas na plataforma são automaticamente sincronizadas com este repositório.

**Via IDE local**

Clone o repositório, faça as alterações desejadas e envie um push. As mudanças também serão refletidas no Lovable.

**Diretamente no GitHub**

Navegue até o arquivo desejado, clique no ícone de lápis (editar) e faça o commit das alterações.

## Sobre a ONG Anjinhos Peludos

A **Anjinhos Peludos** é uma organização não governamental dedicada ao resgate, cuidado e adoção responsável de animais. Este projeto busca ampliar o alcance da ONG por meio de uma plataforma digital acessível e funcional.
